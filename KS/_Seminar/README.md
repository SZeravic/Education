## How to run demo

1. Enter Directory `/public_html`.

2. Install needed modules:

   ```console
   > npm install
   ```

3. In same Directory, Run _development_ server:

   ```console
   > npm start
   ```

   Alternativno:

   ```console
   > gatsby develop
   ```
